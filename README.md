# AI Sales Automation & CRM System

A complete **AI-powered Sales Automation and CRM System** built using **HTML, CSS, JavaScript, and n8n**. This project demonstrates how businesses can automate lead capture, AI-based lead qualification, CRM management, email automation, and customer follow-ups through intelligent workflows.

---

##  Project Overview

This repository contains two integrated modules:

### 📂 1. Sales Automation

An AI-powered lead automation system that captures customer inquiries, validates data, scores leads, automates follow-ups, and sends notifications.

### 📂 2. AI CRM

A lightweight CRM solution that manages customer leads, stores lead information, classifies prospects using AI, and automates customer communication.

Together, these modules demonstrate a complete sales pipeline—from lead generation to customer engagement.
